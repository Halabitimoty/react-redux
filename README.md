# REDUX

A Complex State Management Tool,
with a single store as CDS

# REDUCERS

Manages the State and Returns the
newly updated state

# ACTIONS

Actions have 2 properties type: which
is a unique identifier and payload
which has data.

# DISPATCH

Dispatch is used to send actions to
update the data
